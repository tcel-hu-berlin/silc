Gleichungen:

  • Dissoziationskonstante für Säuren: Ks =[A-][H3O+] / [HA]
  • Dissoziationskonstante für Basen: Kb =[BH+][OH-] / [B]
  • Logarithmische Form: pKx = -log10(Kx)
  • Logarithmus eines Produkts: log(a·b) = log(a) + log(b)
  • Logarithmus eines Bruches: log(a/b) = log(a) - log(b)

  Fakultative Hinweise

Frage 2Einblenden

Versuchen Sie, das Massenwirkungsgesetz für dieses System zuerst aufzustellen, bevor Sie es in die logarithmische Form umwandeln. Um die gewünschte Gleichung herzuleiten muss auch eine Annäherung vorgenommen werden und zwar, dass der Dissoziationsgrad x viel kleiner als die Anfangskonzentration [HA]0 ist.

Frage 7aEinblenden

Denken Sie zunächst an Ihre Ziellösung, wobei die Konzentrationen der Henderson-Hasselbalch-Gleichung entsprechen sollten. Es gibt unendlich viele Lösungen für dieses Problem, je nachdem, wie konzentriert Sie Ihren Puffer haben wollen. Wenn Sie verwirrt sind, setzen Sie Ihre Ziellösung mit 0,100 M Essigsäure und der entsprechenden Menge Acetat an (die Sie berechnen sollen)!

Frage 7bEinblenden

Dieses Problem ist mathematisch komplizierter, da jegliches Acetat im Puffer durch Reaktion von NaOH und Essigsäure hergestellt werden muss.

Frage 8aEinblenden

Behandeln Sie das Problem einfach wie eine Lösung einer schwachen Säure.

Frage 8cEinblenden

Bei der Berechnung des pH-Werts am Äquivalenzpunkt kann das System als Lösung einer schwachen Base behandelt werden, wobei pKB bekannt ist. Achten Sie auf die Änderung des gesamten Volumens der Lösung.

Frage 9bEinblenden

Überlegen Sie sorgfältig, wie ein pKb-Wert in einer Titrationskurve dargestellt wird.

Simulation C: wirksame und unwirksame Puffersysteme

Hier wird wieder eine Lösung aus einer gewünschten Menge von Säure (HA) und konjugierter Base (A-) hergestellt. In der hergestellten Lösung kann dann eine kleine Menge von Natriumhydroxid (NaOH) hinzugefügt werden.

Überblick der Simulation:

  • Die Ausgangskonzentrationen der Säure ([HA]i) und der Base ([A-]i) können gewählt werden.
  • Durch ein Tippen auf Run wird mit der Erstellung der Lösung begonnen.
  • Wenn die Lösung ein Gleichgewicht erreicht, wird mit einem Tippen auf +BASE 5 mL von 5M NaOH hinzugefügt.

Zu Beachten:

  • Die Konzentrationen von HA und A- lassen sich zwischen 0-1 M einstellen.
    Das ist ein anderes Konzentrationsbereich als bei Simulation B!
  • Bitte beachten Sie, dass dieses Programm Punkte (nicht Kommas) als Dezimaltrennzeichen verwendet!!
  • Das Volumen der Lösung kann als 1 L angenommen werden.
