Tipps zur Konstruktion von Lewis-FormelnEinblenden

  1. Bestimmen Sie die Gesamtzahl der Valenzelektronen.
  2. Bestimmen Sie das Zentralatom, das in der Regel das am wenigsten elektronegative Atom ist.
  3. Schreiben Sie eine "Gerüststruktur" des Moleküls, in der das Zentralatom durch Einfachbindungen an die anderen Atome gebunden ist.
  4. Verwenden Sie je zwei Valenzelektronen, um jede Bindung in der "Gerüststruktur" zu bilden.
  5. Versuchen Sie, die Oktettregel für alle Atome zu erfüllen, indem Sie die restlichen Valenzelektronen als Nichtbindungselektronen und Mehrfachbindungen verteilen.

Simulation C: VSEPR-Simulation von PhET™

Diese Simulation ist Teil einer umfangreichen Sammlung von Simulationen für den MINT-Unterricht, die vom PhET™-Team an der Universität von Colorado entwickelt wurde. Mit dieser Simulation können Molekülgeometrien für verschiedene Bindungskonfigurationen um ein Zentralatom betrachtet werden.

Überblick:

  • Für die Aufgaben dieser Aktivität sollte hauptächlich das "Modell"-Panel verwendet werden.
  • Mit Hilfe der Tafeln auf der rechten Seite können bis zu sechs Bindungen oder nichtbindete Elektronenpaare zum Zentralatom hinzugefügt werden.

Zu Beachten:

  • Die Simulation kann auch auch Bindungswinkel und Namen von Geometrien anzeigen. Sie sollten jedoch vermeiden, diese Funktionen übermäßig zu nutzen, damit Sie Ihr eigenes Gedächtnis für diese Konzepte testen und weiterentwickeln können.
