Simulation D: Chemisches Gleichgewicht

Hier kann der Verlauf von zwei Reaktionen beobachtet und die Anzahl der beteiligten Molekülen bzw. Ionen gezählt werden.

Überblick der Simulationen:

  • Eine Animation (links) zeigt je eine Darstellung der beiden Reaktionen auf Partikelebene.
  • Ein begleitendes Diagramm verfolgt die Konzentration der Partikel in Abhängigkeit von der Zeit.
  • Die Animationen können jederzeit mit einer Berührung angehalten bzw. fortgesetzt werden.
  • Wenn eine Animation gestoppt ist, können die genauen Werte im Diagramm durch Berühren abgelesen werden.

Zu Beachten:

  • Jedes Teilchen in den Animationen kann als 1 μmol von Teilchen betrachtet werden.
  • Sie können die Leistung jeder Simulation verbessern, indem Sie sicherstellen, dass die andere Simulation angehalten wird.
